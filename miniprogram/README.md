# 蜂窝陶瓷模具管理微信小程序

## 项目介绍

蜂窝陶瓷模具管理微信小程序是基于WEI设计风格开发的，用于管理蜂窝陶瓷模具的生产、加工、维护等任务。

## 技术栈

- 微信小程序原生开发
- WEI设计风格
- JavaScript
- CSS Variables

## 项目结构

```
miniprogram/
├── app.js              # 小程序入口文件
├── app.json            # 小程序全局配置
├── app.wxss            # 小程序全局样式
├── project.config.json # 小程序项目配置
├── components/         # 自定义组件
│   ├── button/         # 按钮组件
│   └── input/          # 输入框组件
├── pages/              # 页面文件
│   ├── index/          # 首页
│   ├── tasks/          # 任务列表
│   ├── task-detail/    # 任务详情
│   └── profile/        # 个人中心
├── utils/              # 工具类
│   └── api.js          # API请求工具
└── assets/             # 静态资源
    └── icons/          # 图标资源
```

## 快速开始

### 1. 安装微信开发者工具

从[微信开发者工具官网](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)下载并安装微信开发者工具。

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择项目目录：`d:/trae/AT-MJ-3/miniprogram`
4. 填写AppID（如果没有，可以使用测试号）
5. 点击"导入"

### 3. 配置服务器地址

在`app.js`中修改`baseUrl`为你的后端服务器地址：

```javascript
globalData: {
  // ...
  baseUrl: 'http://your-backend-url/api'
}
```

### 4. 运行项目

点击微信开发者工具中的"编译"按钮，即可在模拟器中运行小程序。

## 开发规范

### 1. 目录结构

- 页面文件放在`pages/`目录下，每个页面一个子目录
- 组件文件放在`components/`目录下，每个组件一个子目录
- 工具类放在`utils/`目录下
- 静态资源放在`assets/`目录下

### 2. 命名规范

- 文件名采用小写字母，多个单词用连字符分隔
- 组件名采用大驼峰命名法
- 变量名采用小驼峰命名法
- 常量名采用全大写，多个单词用下划线分隔

### 3. 样式规范

- 使用CSS Variables定义样式变量
- 样式类名采用BEM命名规范
- 组件样式使用scoped属性

## 优化建议

1. **图片优化**：
   - 压缩图片大小
   - 使用WebP格式
   - 延迟加载非首屏图片

2. **性能优化**：
   - 减少setData调用次数
   - 避免在wxml中使用复杂的表达式
   - 使用virtual-list处理长列表

3. **用户体验优化**：
   - 添加加载动画
   - 添加空状态提示
   - 添加下拉刷新和上拉加载
   - 添加错误处理和提示

4. **安全性优化**：
   - 数据加密传输
   - 接口权限验证
   - 防止XSS攻击
