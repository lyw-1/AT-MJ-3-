### 4. 恢复步骤

#### 步骤1：构建后端项目
1. 进入后端项目目录：d:\trae\AT-MJ-3\backend\backend\
2. 运行Maven构建命令：`mvn clean package -DskipTests`
3. 验证jar包生成：检查target目录是否生成jar文件

#### 步骤2：启动后端服务
1. 运行Spring Boot应用：`java -jar target/mold-process-digitalization-1.0.0.jar`
2. 验证服务启动：检查控制台输出，确认服务在端口8082启动

#### 步骤3：构建前端项目
1. 进入前端项目目录：d:\trae\AT-MJ-3\frontend\frontend\
2. 安装依赖：`npm install`
3. 构建项目：`npm run build`
4. 验证构建结果：检查dist目录是否生成静态文件

#### 步骤4：配置Nginx反向代理
1. 编辑Nginx配置文件：D:\Program Files\Apache\Nginx\conf\nginx.conf
2. 添加反向代理配置，将/api请求转发到后端8082端口
3. 配置静态文件路径，指向前端dist目录
4. 重启Nginx服务

#### 步骤5：测试项目运行
1. 访问前端页面：http://localhost
2. 验证登录功能：使用管理员账户登录
3. 验证数据访问：检查模具、设备等数据是否正常显示

### 5. 验证标准
