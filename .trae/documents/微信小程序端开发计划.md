## 预设置页面打印样式优化计划

### 需求分析

1. 预设置页面需要打印为A4纸大小
2. 按A4大小进行页面布局
3. 打印效果要求所有字段一行排列

### 技术实现方案

1. **添加打印样式**：

   * 在 `ProcessPresetPage.vue` 中添加 `@media print` 样式规则

   * 设置A4纸大小：`width: 210mm; height: 297mm;`

   * 调整页面边距：`margin: 0; padding: 1cm;`

   * 确保页面内容居中显示

2. **优化表格布局**：

   * 调整表格宽度为100%，适应A4纸宽度

   * 设置表格单元格不换行：`white-space: nowrap;`

   * 调整字体大小，确保所有字段能在一行显示

   * 优化表格单元格内边距和边框样式

3. **隐藏不必要的元素**：

   * 打印时隐藏左侧工序列表

   * 打印时隐藏顶部导航栏和按钮

   * 只保留主要内容区域

4. **确保所有字段一行排列**：

   * 调整表格列宽，确保所有字段在一行显示

   * 使用 `table-layout: fixed;` 固定表格布局

   * 调整输入框宽度，确保适应表格单元格

### 预期效果

1. 页面在打印预览时显示为A4纸大小
2. 所有字段在打印时一行排列，不换行
3. 打印内容清晰，布局合理
4. 符合用户的打印需求

### 测试方法

1. 在浏览器中打开预设置页面
2. 按 `Ctrl+P` 打开打印预览
3. 检查页面是否为A4纸大小
4. 检查所有字段是否一行排列
5. 检查打印效果是否符合要求
