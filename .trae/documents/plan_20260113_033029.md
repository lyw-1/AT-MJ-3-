# 背景色统一修改计划

## 1. 目标
将WEB端和微信小程序端的背景色统一改为白色（#FFFFFF）

## 2. 修改内容

### 2.1 WEB端修改

#### 文件1：`frontend/frontend/src/styles/variables.scss`
- **修改项**：将 `$bg-color` 从 `#F9F9FA` 改为 `#FFFFFF`
- **影响范围**：整个WEB应用的页面背景色

#### 文件2：`frontend/frontend/src/styles/theme.scss`
- **修改项**：将 `--el-bg-color-page` 从 `#{$gray-50}` 改为 `#{$white}`
- **影响范围**：Element Plus组件库的页面背景色

### 2.2 微信小程序端修改

#### 文件：`miniprogram/app.wxss`
- **修改项**：将 `page` 选择器的 `background-color` 从 `#F9F9FA` 改为 `#FFFFFF`
- **影响范围**：所有小程序页面的背景色

## 3. 技术要点

- **颜色一致性**：确保使用相同的白色值（#FFFFFF）
- **全局修改**：通过修改全局样式变量，实现一次性修改所有页面
- **组件兼容**：确保修改后不影响其他组件的正常显示

## 4. 预期效果

- WEB端所有页面背景色变为白色
- 微信小程序端所有页面背景色变为白色
- 保持其他样式不变
- 确保UI元素（如卡片、按钮等）在白色背景下依然清晰可见

## 5. 验证步骤

1. 修改完成后，启动WEB应用，检查页面背景色
2. 启动微信小程序，检查页面背景色
3. 验证各功能模块正常运行
4. 验证UI元素显示正常

## 6. 风险评估

- **低风险**：仅修改背景色，不涉及核心功能
- **兼容性**：白色背景是最通用的背景色，兼容性良好
- **可恢复性**：如出现问题，可快速恢复到原始值

## 7. 实施时间

- 预计实施时间：10分钟
- 验证时间：5分钟
