# 界面视觉设计规范

## 1. 概述

本设计规范文档旨在为蜂窝陶瓷模具管理系统提供统一的视觉设计标准，确保系统在不同设备上具有一致的视觉风格和用户体验。文档涵盖了色彩系统、排版系统、间距系统、圆角系统、阴影系统等核心设计要素，为开发团队提供清晰的设计指导。

设计规范基于现有的设计系统，结合蜂窝陶瓷模具管理系统的特点和业务需求，确保设计既符合现代UI设计趋势，又满足工业管理系统的实用性要求。

## 2. 设计理念

### 2.1 核心设计原则

- **专业感**：体现工业管理系统的专业性和可靠性
- **简洁性**：保持界面简洁清晰，减少视觉干扰
- **一致性**：在整个系统中保持一致的设计语言
- **易用性**：设计直观易用，减少用户学习成本
- **响应式**：适配不同屏幕尺寸，提供一致的用户体验

### 2.2 设计风格

- **工业风格与现代设计结合**：融合工业风格的专业性和现代设计的简洁性
- **紫色系主题**：使用紫色作为主色调，传达创新和专业感
- **清晰的视觉层次**：通过颜色、大小、间距等元素创建清晰的视觉层次
- **适度的动效**：添加适度的过渡动画，提升用户体验

## 3. 色彩系统

色彩是视觉设计的核心要素，直接影响用户的情绪和体验。本系统采用紫色作为主色调，传达创新和专业感，同时配合中性色和辅助色，创建和谐的色彩体系。

- 主色调主要用于关键交互元素，如按钮、链接、高亮文本等
- 辅助色主要用于状态提示，如成功、警告、错误等
- 中性色主要用于背景、文本、边框等基础元素
- 保持色彩对比度，确保文本可读性
- 避免使用过多色彩，保持界面简洁

## 5. 间距系统

间距系统是创建清晰布局的基础，通过统一的间距单位，确保界面元素之间具有和谐的视觉关系。

### 5.1 间距单位

系统采用8px作为基础间距单位，所有间距均为8px的倍数，确保间距的一致性和和谐性。

| 间距名称 | 尺寸 | 应用场景 |
|---------|------|---------|
| 极小间距 | 8px | 组件内部元素间距 |
| 小间距 | 16px | 表单元素间距、卡片内部间距 |
| 中间距 | 24px | 模块间距、卡片间距 |
| 大间距 | 32px | 页面区域间距 |
| 极大间距 | 40px | 主要模块间距 |
| 超大间距 | 48px | 页面大区域间距 |

### 5.2 间距应用场景

| 应用场景 | 建议间距 |
|---------|---------|
| 页面内边距 | 24px |
| 卡片内边距 | 24px |
| 表单字段间距 | 16px |
| 按钮间距 | 16px |
| 模块间距 | 32px |
| 列表项间距 | 16px |
| 图标与文本间距 | 8px |

### 5.3 间距应用原则

- 使用统一的间距单位，保持界面一致性
- 建立清晰的视觉层次，通过间距区分不同模块
- 保持适当的留白，避免界面过于拥挤
- 响应式调整间距，在小屏幕上减少间距，提高内容密度

## 10. 响应式设计规范

响应式设计确保系统在不同屏幕尺寸上具有良好的显示效果和用户体验。

### 10.1 断点设计

| 断点名称 | 屏幕宽度 | 设备类型 | 布局调整 |
|---------|---------|---------|---------|
| 超小屏幕 | < 576px | 手机 | 单列布局，底部导航 |
| 小屏幕 | 576px - 768px | 平板 | 双列布局，侧边栏导航 |
| 中等屏幕 | 768px - 992px | 小屏电脑 | 双列布局，完整侧边栏导航 |
| 大屏幕 | 992px - 1200px | 中屏电脑 | 三列布局，完整侧边栏导航 |
| 超大屏幕 | 1200px - 1600px | 大屏电脑 | 三列布局，完整侧边栏导航 |
| 巨幕屏幕 | >= 1600px | 超大屏电脑 | 三列布局，完整侧边栏导航 |

### 10.2 响应式布局原则

- **移动优先设计**：从移动设备开始设计，然后扩展到更大的屏幕
- **流体布局**：使用相对单位，确保内容能够根据屏幕尺寸自动调整
- **弹性网格**：使用栅格系统，允许内容在不同屏幕尺寸下重新排列
- **内容优先级**：在小屏幕上只显示核心内容，次要内容可隐藏或通过交互访问
- **触控友好**：确保交互元素有足够的点击区域（至少44x44像素）

### 10.3 响应式组件调整

| 组件 | 大屏幕调整 | 小屏幕调整 |
|------|-----------|-----------|
| 导航 | 完整侧边栏导航 | 底部导航或汉堡菜单 |
| 表格 | 完整列显示 | 隐藏次要列，使用垂直堆叠布局 |
| 表单 | 多列布局 | 单列表单布局 |
| 卡片 | 多列布局 | 单列布局 |
| 图表 | 完整图表 | 简化图表，只显示核心数据 |
