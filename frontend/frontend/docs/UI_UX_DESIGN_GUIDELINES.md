# UI/UX设计规范文档

## 1. 概述

本设计规范基于AT-MJ-3系统HOME页面的设计风格，旨在为整个系统提供统一的视觉设计和用户体验标准。所有规范均与HOME页面主区域的设计风格保持高度一致，确保系统各页面之间的视觉连贯性和用户体验一致性。

## 3. 排版体系

### 3.1 字体家族

- 主字体：系统默认无衬线字体（如San Francisco, Roboto, Segoe UI）
- 备选字体：'Helvetica Neue', Arial, sans-serif

### 3.2 字号层级

| 字号 | 名称 | 应用场景 |
|------|------|----------|
| 12px ($font-size-xs) | 超小 | 次要标签、状态文本、元数据 |
| 14px ($font-size-sm, $font-size-base) | 小/基础 | 正文文本、卡片内容、按钮文本 |
| 16px ($font-size-lg) | 大 | 卡片标题、模块标题、导航项 |
| 18px ($font-size-xl) | 超大 | 页面副标题、重要标题 |
| 20px ($font-size-2xl) | 特大 | 页面主标题、欢迎语标题 |
| 36px | 统计值 | 统计卡片数值、关键指标 |

### 3.3 字重规范

| 字重 | 名称 | 应用场景 |
|------|------|----------|
| 400 | 常规 | 正文文本、普通内容 |
| 500 ($font-weight-medium) | 中等 | 按钮文本、卡片标题、重要内容 |
| 600 ($font-weight-semibold) | 半粗 | 模块标题、强调文本、统计标签 |
| 700 | 粗体 | 页面主标题、统计数值、重要强调 |

### 3.4 行高设置

| 字号 | 行高 | 应用场景 |
|------|------|----------|
| 12px | 1.4 | 超小文本 |
| 14px | 1.5 | 小/基础文本 |
| 16px | 1.5 | 大文本 |
| 18px | 1.6 | 超大文本 |
| 20px | 1.6 | 特大文本 |
| 36px | 1.2 | 统计数值 |

### 3.5 段落间距

| 元素类型 | 间距 |
|----------|------|
| 标题与正文 | 8px |
| 正文段落 | 16px |
| 列表项 | 8px |
| 卡片内元素 | 8px |
| 模块间间距 | 16px |

## 4. 组件规则

### 4.1 卡片组件

**基础卡片**
- 背景色：#FFFFFF
- 边框：1px solid #E5E7EB
- 圆角：15px
- 阴影：0 4px 12px rgba(0, 0, 0, 0.15)
- 悬停效果：阴影增大至0 8px 24px rgba(0, 0, 0, 0.25)，上移1-3px
- 内边距：24px ($spacing-3)
- 外边距：8-15px

**统计卡片**
- 固定高度：120px
- 渐变背景：使用预设渐变色彩
- 文本颜色：#FFFFFF
- 数值大小：36px
- 标签大小：14px
- 圆角：15px
- 阴影：0 4px 12px rgba(0, 0, 0, 0.15)
- 悬停效果：阴影增大至0 8px 24px rgba(0, 0, 0, 0.25)，上移3px

**快捷入口卡片**
- 最小高度：50px
- 背景色：#FFFFFF
- 边框：1px solid #E5E7EB
- 圆角：15px
- 阴影：0 2px 8px rgba(0, 0, 0, 0.06)
- 悬停效果：阴影增大至0 4px 16px rgba(0, 0, 0, 0.08)，边框颜色变为#6150BF，上移1px
- 内边距：8px 16px
- 文字大小：14px

### 4.2 按钮样式

- 背景色：透明

**主按钮**

- 文字大小：14px
- 悬停效果：背景色变为#4F46E5，阴影增大

**状态按钮**
- 成功状态：背景色#D1FAE5，文字颜色#10B981

- 错误状态：背景色#FECACA，文字颜色#EF4444

### 4.3 表单元素

**输入框**
- 背景色：#FFFFFF
- 边框：1px solid #D1D5DB
- 圆角：12px ($border-radius-sm)
- 内边距：12px
- 文字大小：14px
- 悬停效果：边框颜色变为#9CA3AF

**选择器**

- 下拉箭头：使用Element Plus默认图标
- 选项背景：#FFFFFF，悬停变为#F3F4F6

### 4.4 导航元素

**侧边栏导航**
- 背景色：#FFFFFF
- 文字颜色：#6B7280
- 激活状态：文字颜色#6150BF，背景色#EEF2FF
- 悬停效果：背景色#F3F4F6，文字颜色#4B5563
- 菜单项高度：48px
- 图标大小：18px

**面包屑导航**
- 文字颜色：#6B7280
- 分隔符："/"，颜色#9CA3AF
- 文字大小：14px
- 内边距：0 16px

## 5. 布局标准

### 5.1 网格系统

- 基础网格：24列布局（基于Element Plus）
- 响应式断点：
  - 超小屏幕 (<576px)：1列
  - 小屏幕 (≥576px)：2列
  - 中等屏幕 (≥768px)：3-4列
  - 大屏幕 (≥992px)：4-6列
  - 超大屏幕 (≥1200px)：6-8列

### 5.2 边距规范

| 元素类型 | 边距值 |
|----------|--------|
| 页面容器 | 15px |
| 模块间距 | 10px |
| 卡片间距 | 15px |
| 卡片内边距 | 24px ($spacing-3) |
| 快捷入口间距 | 15px |
| 列表项间距 | 8px |

### 5.3 容器尺寸

| 容器类型 | 尺寸 |
|----------|------|
| 统计卡片 | 固定高度120px |
| 快捷入口卡片 | 最小高度50px |
| 功能模块卡片 | 最小高度250px |
| 页面主容器 | 100% 宽度，最小高度 calc(100vh - 120px) |

### 5.4 响应式断点

| 断点名称 | 屏幕宽度 | 布局调整 |
|----------|----------|----------|
| xs | <576px | 单列布局，减少内边距，简化导航 |
| sm | ≥576px | 双列布局，恢复标准内边距 |
| md | ≥768px | 三列布局，显示完整导航 |
| lg | ≥992px | 四列布局，扩展内容区域 |
| xl | ≥1200px | 六列布局，最大化内容展示 |

## 6. 交互模式

### 6.1 悬停效果

- **卡片悬停**：阴影增大，轻微上移（1-3px）
- **按钮悬停**：背景色变化，边框颜色加深
- **列表项悬停**：背景色变为#F3F4F6，轻微上移
- **快捷入口悬停**：边框颜色变为#6150BF，阴影增大

### 6.2 过渡动画

- **卡片悬停**：0.3s 缓动效果
- **按钮状态变化**：0.2s 缓动效果
- **页面加载**：渐入效果（0.5s）
- **模态框**：淡入淡出 + 缩放效果
- **下拉菜单**：淡入 + 上滑效果

### 6.3 反馈机制

- **成功操作**：绿色提示框，显示成功信息
- **错误操作**：红色提示框，显示错误信息
- **警告信息**：黄色提示框，显示警告信息
- **加载状态**：使用Element Plus加载图标，显示在操作按钮或页面中心
- **点击反馈**：按钮按下状态，轻微缩放

### 6.4 加载状态

- **页面加载**：全屏加载动画，显示在页面中心
- **组件加载**：局部加载动画，显示在组件中心
- **数据加载**：骨架屏或加载图标，显示在数据区域
- **按钮加载**：按钮内加载图标，禁用按钮交互

## 8. 设计规范一致性

为确保整个系统的视觉风格和用户体验统一连贯，所有页面和组件设计必须严格遵循本规范。在设计新页面或组件时，应：

1. 优先使用规范中定义的色彩、排版和组件样式
2. 保持与HOME页面主区域的设计风格高度一致
3. 遵循相同的交互模式和反馈机制
4. 确保响应式设计在所有断点下正常工作
5. 使用相同的阴影、圆角和间距规范

## 9. 维护与更新

本设计规范将随着系统的发展和用户反馈不断更新和完善。每次更新应记录变更内容、原因和影响范围，并通知所有相关设计和开发人员。

## 10. 总结

本UI/UX设计规范文档为AT-MJ-3系统提供了全面的设计指导，确保系统各页面之间的视觉连贯性和用户体验一致性。所有规范均基于HOME页面主区域的设计风格，包含具体的数值、示例和应用说明，具备高度的可操作性。

通过严格遵循本规范，设计和开发团队可以快速、一致地创建出符合系统整体风格的页面和组件，从而提高开发效率，降低设计和开发成本，同时为用户提供统一、流畅的使用体验。
