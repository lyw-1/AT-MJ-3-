# Clarity Design Aone 基础组件库 - UI规范文档

## 1. 概述

本设计规范基于Pixso设计文件「Clarity Design Aone 基础组件库（来自社区）」，旨在为AT-MJ-3系统提供统一的视觉设计和用户体验标准。所有规范均与Clarity Design组件库的设计风格保持高度一致，确保系统各页面之间的视觉连贯性和用户体验一致性。

## 6. 组件样式规范

### 6.1 卡片组件

**基础卡片**
- 背景色：--bg-card
- 边框：1px solid --color-gray-200
- 圆角：--radius-card
- 阴影：0 2px 8px rgba(0, 0, 0, 0.04)
- 内边距：--space-3
- 外边距：--space-2

**卡片变体**
- 统计卡片：固定高度120px，渐变背景，文本颜色--color-white
- 快捷入口卡片：最小高度50px，简洁设计，悬停效果明显

### 6.2 按钮样式

**主按钮**
- 背景色：--color-primary
- 文字颜色：--color-white
- 圆角：--radius-button
- 内边距：8px 16px
- 文字大小：--font-size-base
- 悬停效果：背景色变为--color-primary-dark
- 点击效果：transform: translateY(1px)

**次要按钮**
- 背景色：--color-white
- 边框：1px solid --color-gray-200
- 文字颜色：--color-gray-700
- 圆角：--radius-button
- 内边距：8px 16px
- 文字大小：--font-size-base
- 悬停效果：背景色变为--color-gray-50

**状态按钮**
- 成功状态：背景色#D1FAE5，文字颜色--color-success
- 警告状态：背景色#FEF3C7，文字颜色--color-warning
- 错误状态：背景色#FECACA，文字颜色--color-error
- 尺寸：4px 8px圆角，文字大小12px

### 6.3 表单元素

**输入框**
- 背景色：--color-white
- 边框：1px solid --color-gray-300
- 圆角：--radius-input
- 内边距：12px
- 文字大小：--font-size-base
- 悬停效果：边框颜色变为--color-gray-400
- 聚焦效果：边框颜色变为--color-primary，添加2px阴影

**选择器**
- 与输入框样式一致
- 下拉箭头：使用Element Plus默认图标
- 选项背景：--color-white，悬停变为--color-gray-50

### 6.4 导航元素

**侧边栏导航**
- 背景色：--color-white
- 文字颜色：--color-gray-700
- 激活状态：文字颜色--color-primary，背景色--color-primary-light
- 悬停效果：背景色变为--color-primary-light，文字颜色变为--color-primary
- 菜单项高度：48px
- 图标大小：18px

**面包屑导航**
- 文字颜色：--color-gray-600
- 分隔符："/"，颜色--color-gray-400
- 文字大小：--font-size-sm
- 内边距：0 16px

### 6.5 表格样式

**基础表格**
- 背景色：--color-white
- 边框：1px solid --color-gray-200
- 圆角：--radius-lg
- 阴影：0 2px 8px rgba(0, 0, 0, 0.06)

**表格头部**
- 背景色：--color-white
- 文字颜色：--color-gray-700
- 字体权重：--font-weight-semibold
- 高度：52px
- 边框底部：2px solid --color-gray-200

**表格行**
- 高度：48px
- 边框底部：1px solid --color-gray-100
- 悬停效果：背景色变为--color-gray-50

### 6.6 标签样式

**基础标签**
- 圆角：--radius-lg
- 内边距：4px 12px
- 字体大小：--font-size-sm
- 字体权重：--font-weight-medium
- 边框：none

**标签变体**
- 主标签：背景色--color-primary-light，文字颜色--color-primary
- 成功标签：背景色rgba(16, 185, 129, 0.1)，文字颜色--color-success
- 警告标签：背景色rgba(245, 158, 11, 0.1)，文字颜色--color-warning

## 7. 交互模式

### 7.1 悬停效果

- **卡片悬停**：阴影增大，轻微上移（1-3px）
- **按钮悬停**：背景色变化，边框颜色加深
- **列表项悬停**：背景色变为--color-gray-50，轻微上移
- **快捷入口悬停**：边框颜色变为--color-primary，阴影增大

### 7.2 过渡动画

- **卡片悬停**：0.3s 缓动效果
- **按钮状态变化**：0.2s 缓动效果
- **页面加载**：渐入效果（0.5s）
- **模态框**：淡入淡出 + 缩放效果
- **下拉菜单**：淡入 + 上滑效果

### 7.3 反馈机制

- **成功操作**：绿色提示框，显示成功信息
- **错误操作**：红色提示框，显示错误信息
- **警告信息**：黄色提示框，显示警告信息
- **加载状态**：使用Element Plus加载图标，显示在操作按钮或页面中心
- **点击反馈**：按钮按下状态，轻微缩放

### 7.4 加载状态

- **页面加载**：全屏加载动画，显示在页面中心
- **组件加载**：局部加载动画，显示在组件中心
- **数据加载**：骨架屏或加载图标，显示在数据区域
- **按钮加载**：按钮内加载图标，禁用按钮交互

## 8. 响应式设计

### 8.1 断点规范

| 断点名称 | 屏幕宽度 | 布局调整 |
|----------|----------|----------|
| xs | <576px | 单列布局，减少内边距，简化导航 |
| sm | ≥576px | 双列布局，恢复标准内边距 |
| md | ≥768px | 三列布局，显示完整导航 |
| lg | ≥992px | 四列布局，扩展内容区域 |
| xl | ≥1200px | 六列布局，最大化内容展示 |

### 8.2 响应式调整原则

1. **内容优先**：在小屏幕上优先展示核心内容
2. **简化导航**：在小屏幕上简化导航结构，可折叠
3. **调整间距**：在小屏幕上适当减小间距，提高信息密度
4. **调整字体大小**：在小屏幕上保持合适的字体大小，确保可读性
5. **调整组件大小**：在小屏幕上调整组件尺寸，确保触摸友好

## 9. 设计规范一致性

为确保整个系统的视觉风格和用户体验统一连贯，所有页面和组件设计必须严格遵循本规范。在设计新页面或组件时，应：

1. 优先使用规范中定义的色彩、排版和组件样式
2. 保持与Clarity Design组件库的设计风格高度一致
3. 遵循相同的交互模式和反馈机制
4. 确保响应式设计在所有断点下正常工作
5. 使用相同的阴影、圆角和间距规范

## 10. 维护与更新

本设计规范将随着系统的发展和用户反馈不断更新和完善。每次更新应记录变更内容、原因和影响范围，并通知所有相关设计和开发人员。

## 11. 总结

本Clarity Design UI规范文档为AT-MJ-3系统提供了全面的设计指导，确保系统各页面之间的视觉连贯性和用户体验一致性。所有规范均基于Pixso设计文件「Clarity Design Aone 基础组件库（来自社区）」，包含具体的数值、示例和应用说明，具备高度的可操作性。

通过严格遵循本规范，设计和开发团队可以快速、一致地创建出符合系统整体风格的页面和组件，从而提高开发效率，降低设计和开发成本，同时为用户提供统一、流畅的使用体验。
